# OPEN-USB-PS-V3

#### 介绍
USB可调电源V3,基于雅特力的MCU AT32F421F8P7和杰里的BLE蓝牙芯片AC6328开发，
输入电压范围：DC5V~27V
输出电压范围：DC2.0V-34V
输出电流能力>5A
输出限流范围：10mA~8A

#### 软件架构
个人开发的mcu_os 软件框架
